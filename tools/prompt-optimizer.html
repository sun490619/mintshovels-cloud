💡 Pro Tips for Better Prompts
1. Be Specific
Instead of "write about AI", say "write a 500-word introduction to transformer models for beginners"
2. Define Format
Specify output format: bullet points, JSON, markdown, table, or code blocks
3. Give Context
"You are a senior engineer reviewing this React code for performance issues"
4. Use Examples
Provide input/output examples to guide the model's response style
