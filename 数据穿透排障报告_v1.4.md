### 修复

## 断层 2️⃣：`tools/*.html` 静态页面是否全量重刷？

### 排查过程
- `engine/auto_factory.py` 存在 `generate_tool_page()` 函数
- 但**没有** `regenerate_all()` / `batch_render()` 批量重渲染函数
- `tools/` 目录只有 39 个文件

### 发现
```
tools/ 目录:
  总HTML文件: 39个
  手写工具页面: 33个
  自动工具页面: 0个（全部动态渲染）
```

**结论**: 自动工具**不生成独立HTML页面**。前端通过 API 动态加载：
```javascript
const r = await fetch(API + "/v1/workshop/tools");
AUTO_TOOLS = d.tools.filter(t => t.deployed).map(mapAutoTool);
```
点击工具时再通过 API 获取详情渲染。**不需要重刷 tools/*.html**。
